# CV Website

## Project Info
This is a personal CV website built using **HTML and CSS only**.  
It is responsive on both desktop and mobile screens. The website showcases personal profile, skills, languages, hobbies, education, and projects. It also includes bonus features such as a downloadable CV and hover animations.

## Technologies Used
- HTML
- CSS
- Flexbox

## Bonus Features
- Downloadable CV (PDF)
- Hover animations for buttons and project items
- Responsive design for mobile and desktop
- Neat bullet circles for skills, hobbies, and languages
